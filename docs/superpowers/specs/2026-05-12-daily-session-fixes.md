## Problem Statement

Daily Practice состоит из 3 блоков: переводы (10), словарь (5 флешкарт), глаголы (10 спряжений).
Выявлены 4 категории багов + необходимость документации навигации.

## Task 1: Микрофон и кнопка Start в блоке 1 при первом уроке

### Спецификация

- TTS **НЕ** автовоспроизводится на первой карточке нового урока — это **правильно**
- Кнопка **Start** должна быть активна и реагировать на нажатие
- Иконка **микрофона** должна быть активна и запускать ASR
- Оба элемента управляются через `inputMode` и `isListening` стейт — проверить, что они не блокируются при отсутствии автовоспроизведения

### Критерий готовности

- [ ] На первой карточке нового урока TTS не играет автоматически
- [ ] Кнопка Start нажимается и запускает сессию
- [ ] Иконка микрофона нажимается и запускает ASR-прослушивание
- [ ] После ручного старта — навигация и ввод работают как обычно

## Task 2: Учёт карточек в блоках 1 и 3 — ответ обязательный

### Спецификация

Карточка считается «показанной» (практикованной) **только** если получен ответ через:
- **VOICE** (ASR-распознавание)
- **KEYBOARD** (ручной ввод)

**НЕ** считаются:
- WORD_BANK (выбор из вариантов)
- Простое пролистывание кнопкой «Вперёд»

#### Навигация

- Навигация вручную (кнопка «Вперёд» / свайп) позволяет листать **бесконечно** — это ок
- Если карточка пролистана без ответа VOICE/KEYBOARD — она **не** отмечается как практикованная
- Карточки, забракованные через «плохие карты» (bad words), исключаются из подсчёта

#### Продвижение курсора между сессиями

- Курсор между дейли-сессиями (между днями) продвигается **только** когда все 10 карточек блока (минус забракованные) получили ответ VOICE или KEYBOARD
- Если хоть одна карточка не отвечена — курсор **стоит**
- При повторном входе в тот же блок — показываем те же карточки с начала, не генерируем новые

### Критерий готовности

- [ ] Пролистывание карточки без ответа не отмечает её как практикованную
- [ ] Ответ через VOICE отмечает карточку
- [ ] Ответ через KEYBOARD отмечает карточку
- [ ] Ответ через WORD_BANK НЕ отмечает карточку
- [ ] Навигация вперёд работает бесконечно
- [ ] Курсор НЕ продвигается, если не все карточки отвечены
- [ ] Курсор продвигается, когда все карточки (минус bad words) отвечены
- [ ] При незавершённом блоке — повторный вход показывает те же карточки

## Task 3: Информация о глаголе на карточках блока 3

### Спецификация

Карточки спряжения глаголов в блоке 3 (Verb Drill) должны отображать метаданные глагола:
- **Инфинитив** (глагол в неопределённой форме)
- **Время** (Tense: Present, Past, Future и т.д.)
- **Группа** (Group спряжения, если указана)

Аналогично уже реализованному в экране Verb Drill (`VerbDrillScreen.kt` / `VerbDrillViewModel.kt`).

Формат отображения: над основным переводом — строка типа:
```
parlare — Present, Group 1
```

Данные берутся из VerbDrill CSV: колонки `Verb`, `Tense`, `Group`.

### Критерий готовности

- [ ] На карточках блока 3 отображается инфинитив глагола
- [ ] Отображается время (Tense)
- [ ] Отображается группа (Group) если есть
- [ ] Формат аналогичен Verb Drill экрану
- [ ] Если метаданных нет — карточка отображается без них (не крашится)

## Task 4: Логика навигации между Daily-сессиями

### Спецификация

#### Кнопка «Повтор» (Repeat)

- Показывает **те же карточки**, что были в первой сессии сегодняшнего дня
- Состав карточек хранится временно — **24 часа**
- Используется для повторного прохождения уже виденных карточек
- Прогресс mastery НЕ сбрасывается при повторе (уже ответенные карточки остаются ответенными)

#### Кнопка «Продолжить» (Continue)

- Если курсор продвинулся за первую сессию — показывает **следующую порцию** новых карточек
- Если курсор не продвинулся — эквивалентно «Повтор»
- Для блока 2 (словарь) — слова идут по своему SRS-процессу независимо от блоков 1 и 3

#### Хранение состояния

- Состав карточек первой сессии: `DailyCursorState` + `todayShownCardIds`
- Курсор между сессиями: `sentenceOffset`, `verbOffset` в `DailyCursorState`
- Срок жизни «первых карточек дня»: до полуночи (или 24ч с момента генерации)

### Критерий готовности

- [ ] «Повтор» показывает те же карточки, что были первой сессией сегодня
- [ ] «Продолжить» показывает новые карточки, если курсор продвинулся
- [ ] «Продолжить» работает как «Повтор», если курсор не продвинулся
- [ ] Слова (блок 2) следуют SRS-процессу независимо
- [ ] Состав первых карточек дня хранится 24ч
- [ ] Прогресс mastery не сбрасывается при повторе
